1. Introduction

git-migrate is a command-line tool created with the intent of making the move from one git forge to another more convenient.

Supported forges include:

  • Github
  • Codeberg
  • Forgejo

Forges to add include:

  • Sourcehut
  • Gitlab main
  • Gitlab self-host
  • Gitea
